# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in totp-js, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: pratiyush1@gmail.com
3. Include: description, steps to reproduce, potential impact
4. You will receive a response within 48 hours

## Security Features

totp-js is designed with security as a first-class concern:

- Constant-time comparison to prevent timing attacks
- Built-in replay protection
- Secure random number generation for secrets
- No sensitive data in error messages
